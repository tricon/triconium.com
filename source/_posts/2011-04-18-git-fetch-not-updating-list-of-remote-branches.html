--- layout: post title: Git Fetch Not Updating List of Remote Branches published: true guid: 151 ---

If this is the case, the issue is that your remote is setup as such:

[remote "origin"]
      url = url
      fetch = +refs/heads/master:refs/remotes/origin/master

and instead should be:

[remote "origin"]
      url = url
      fetch = +refs/heads/*:refs/remotes/origin/*

in order to fetch all remote branches, not just master.
