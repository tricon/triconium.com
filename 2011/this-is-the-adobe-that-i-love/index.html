This Is the Adobe That I Love

Adobe Edge:

Adobe® Edge is a new web motion and interaction design tool that allows designers to bring animated content to websites, using web standards like HTML5, JavaScript, and CSS3.

These demos are pretty exciting and show just how well such technology can run in the browser — without Flash.

As an added bonus, the user interface takes after After Effects, which is one of Adobe’s better UIs.
