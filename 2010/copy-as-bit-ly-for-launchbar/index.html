Copy as bit.ly for LaunchBar

Want to quickly make a short URL using bit.ly and use LaunchBar? The script below should help you.

Unzip it and place it in ~/Library/Scripts, open the file, and replace “USERNAME” and “APIKEY” with your bit.ly username and api key. If the script doesn’t show up in LaunchBar, you need to add the Scripts folder to your LaunchBar index. To do so:

  1. Activate LaunchBar
  2. Click the gear
  3. Go to Index –> Show Index
  4. Click the + in the bottom left
  5. Add the folder ~/Library/Scripts

From now on, any scripts you add to that folder will be indexed by LaunchBar.

To use the script:

  1. Activate LaunchBar
  2. Begin typing part of the name of the script
  3. Hit space
  4. Paste your URL
  5. Hit enter Your shortened URL will automatically be copied to the clipboard!
